Coffeescript, qu'est ce donc ?


  • Un langage avec une syntaxe inspirée de ruby et python (entre autre)
  • Corrigeant les problèmes majeurs de javascript (verbosité et consistance)
  • Compilant en javascript ...
  • ... avec une sortie propre et passant la validation JS Lint
square = (x) -> x * x

if [] != 0
   console.log '\o/'
            
var square;

square = function(x) {
    return x * x;
};
if ([] !== 0) {
    console.log('\o/');
}
4/7

Coffeescript, comment ca marche ?


  • Compilation avec l'utilitaire fourni par le package coffee-script
  • Compilation "en continue" avec l'option --watch
  • Utilisation de librairies tierces (Wro, jawr, etc.)
  • Compilation en direct dans le navigateur en incluant un script
  • En copiant/collant le code dans sur le site de coffee : http://coffeescript.org
5/7

Quelques informations supplémentaires


  • Coffeescript utilise les API javascript
  • Coffeescript utilise l'indentation comme syntaxe
  • les fonctions se définissent avec (param) -> corps
  • les attributs d'objets peuvent être récupérés avec @
  • le mot clef return est optionnel
  • les ; sont optionnels
  • les parenthèses sont (la plus part du temps) optionnels
